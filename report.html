1 Introduction

Cette analyse illustre un workflow complet en R (tidyverse + ggplot2 + Quarto) : nettoyage, statistiques descriptives, régression et visualisations propres.

Le dataset Gapminder contient des séries historiques par pays (population, PIB/habitant, espérance de vie) sur plusieurs années.

6 Bonus: performance prédictive (validation simple)

Une validation temporelle minimaliste : entraîner sur les années < dernière année, tester sur la dernière année.

Code 
train <- df %>% filter(year < last_year)
test  <- df %>% filter(year == last_year)

m_train <- lm(life_exp ~ log_gdp_percap + continent + year, data = train)

test_pred <- test %>%
  mutate(pred = predict(m_train, newdata = test),
         err = life_exp - pred)

rmse <- sqrt(mean(test_pred$err^2))
rmse
[1] 6.448048
Code 
p_pred <- ggplot(test_pred, aes(pred, life_exp, color = continent)) +
  geom_point(alpha = 0.7) +
  geom_abline(slope = 1, intercept = 0, linetype = "dashed") +
  labs(
    title = paste0("Prédictions vs réel (", last_year, ")"),
    subtitle = paste0("RMSE = ", round(rmse, 2), " ans (validation temporelle)"),
    x = "Espérance de vie prédite", y = "Espérance de vie observée", color = "Continent"
  )

7 Conclusion

  • L’espérance de vie augmente fortement au fil du temps dans toutes les régions.
  • Le PIB/habitant est fortement associé à l’espérance de vie (relation non-linéaire suggérée, d’où l’usage du log10).
  • Le modèle linéaire avec contrôles (continent, année) fournit une base solide et reproductible, et se prête à des extensions (effets par pays, modèles non-linéaires, etc.).
